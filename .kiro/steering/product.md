# Product Overview

Modbus Connect is a professional Modbus TCP/IP scanner and real-time monitoring tool for industrial automation engineers. The product helps users discover Modbus devices on their network, monitor them in real-time, visualize data, and analyze protocol traffic.

## Target Audience

Industrial automation professionals working with SCADA systems, PLCs, and Modbus devices.

## Key Features

- Smart device discovery (scan device IDs 1-247)
- Real-time monitoring with configurable polling intervals (100ms to 60s)
- Data visualization with real-time charts (up to 12 series)
- Workspace management (save/load configurations)
- Protocol logging (raw TX/RX inspection)
- High performance (Rust backend)

## Platform

Windows desktop application (Windows 7/8/10/11, 64-bit)
